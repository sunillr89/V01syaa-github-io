SIDIA Android-Based VCI Tool for Commercial Vehicle

These Diagnostic Tools have two components. The first one is a Hardware (VCI Tool) that needs to be connected to the Vehicle's OBD Port. The Second part of the Diagnostic system is a Use-Friendly Application. Based on the requirements of the user, we can either provide an Android App or a Desktop App.

The android app is preferred for Wireless communication with the VCI tools over Bluetooth. The android app can be downloaded from the Google Play store and Installed onto an Android Smart Phone. The Android Phone communicates with the Hardware VCI Tool wirelessly over Bluetooth. The User operates the VCI tool from the Android Application.

The Desktop app is developed with Dot Net, which makes it compatible across multiple versions of widely used Windows Operating System. We provide frequent security updates and for new features, as required.

The VCI Tools support both the standard OBD-II Protocols and OEM Specific Diagnostic Functions. The user can use it for Diagnostic Functions, such as checking Diagnostic Trouble Codes (DTCs), Live Parameters (e.g., Engine RPM, Air In-take Temperature, Oxygen Levels, Exhaust Temperatures, etc). The VCI Tools also support ECU Flashing.
